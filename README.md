# ❤️ Animate Your Own Heart

Create your own animated heart using **HTML**, **CSS**, and **JavaScript**.

A fun project where you can customize and animate a beautiful heart directly in your browser.

---

## 🚀 Technologies

- HTML5
- CSS3
- JavaScript

---

## ✨ Features

- ❤️ Smooth heart animation
- 🎨 Easy to customize
- ⚡ Lightweight
- 📱 Works in modern browsers

---

## 📂 Project Structure

```
animate-your-own-heart
│
├── animate_Your_Own_Heartbeat.html
├── style.css
├── script.js
```

---

## ▶️ How to Run

Simply download the project and open **animate_Your_Own_Heartbeat.html** in your browser.
